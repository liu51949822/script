# oracle — 高级咨询代理

## 简介

只读的高 IQ 推理顾问，用于复杂架构设计、疑难调试和方案评审。
适合需要深度分析、方案比选、代码审查的场景。

## 使用场景

- 复杂系统架构设计评审
- 棘手的 Bug 深度分析
- 方案选型和技术决策
- 安全性和性能评估
- 跨系统集成方案设计
- 实现后的自我审查 (替代人工 Code Review)

## 用法

```text
task(subagent_type="oracle", load_skills=[], run_in_background=false,
     prompt="[上下文]: ... [方案]: ... [顾虑]: ... [评估要求]: ...")
```

### 适用场景

| 场景 | 在解决前先咨询 Oracle | 原因 |
|------|----------------------|------|
| 架构决策 | 评估方案优劣 | 避免方向性错误 |
| 2+ 次修复失败 | 分析根因 | 避免反复试错 |
| 安全/性能 | 评审方案 | 确保最佳实践 |
| 大量实现后 | 自我审查 | 发现遗漏问题 |

## 示例

```text
task(subagent_type="oracle", load_skills=[], run_in_background=false,
     prompt="需要架构评审。我的方案：[描述具体方案和文件变更]。
     我的顾虑：[列出具体不确定性]。
     请评估：方案正确性、潜在问题、更好的替代方案。")
```

## 参考

- 类型: subagent_type="oracle"
- 特点: 高智能、偏高成本、只读咨询
- 注意: 应在方案确定前或实现完成后调用
