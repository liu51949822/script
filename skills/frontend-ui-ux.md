# /frontend-ui-ux — 前端 UI/UX 设计技能

## 简介

设计师转开发的 UI 专家，无需设计稿即可产出高质量的前端界面。适合所有前端/UI/UX 相关任务。

## 使用场景

- 用户界面设计和实现
- 响应式布局和适配
- 交互动画和过渡效果
- 组件库搭建
- UI 改版和优化
- 无设计稿时的界面创建

## 用法

所有 UI 相关的任务强制使用此技能：

```text
task(category="visual-engineering", load_skills=["frontend-ui-ux"],
     prompt="创建一个用户仪表盘页面...")
```

### 强制规则

| 任务类型 | 必须使用 |
|----------|----------|
| UI 组件开发 | visual-engineering + frontend-ui-ux |
| 页面布局 | visual-engineering + frontend-ui-ux |
| CSS/Styling | visual-engineering + frontend-ui-ux |
| 动画/过渡 | visual-engineering + frontend-ui-ux |
| 设计实现 | visual-engineering + frontend-ui-ux |
